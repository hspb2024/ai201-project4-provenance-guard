# Provenance Guard

A content-attribution service. A creator submits text; the system estimates whether
it was written by a human or generated by AI, attaches a plain-English transparency
label with a calibrated confidence score, records the decision in a SQLite audit log,
and (from Milestone 5) lets creators appeal a result.

Full design — detection signals, uncertainty model, label variants, appeals workflow,
edge cases, and architecture diagrams — lives in [planning.md](planning.md).

## Status

- **Milestone 1–2:** architecture + spec — see [planning.md](planning.md).
- **Milestone 3 (current):** `POST /submit`, Signal 1 (Groq LLM classifier),
  structured SQLite audit log, `GET /log`. Confidence and label are placeholders
  until Milestones 4–5.

## Detection signals

Two genuinely independent signals (only Signal 1 is wired in M3):

1. **LLM classifier (Groq)** — *semantic/stylistic.* Asks the model for an
   AI-likelihood in `[0,1]` plus a rationale.
2. **Stylometric heuristics (pure Python)** — *structural.* Sentence-length variance,
   type-token ratio, punctuation variety. **(Milestone 4.)**

## Setup

```bash
python -m venv .venv
source .venv/Scripts/activate      # Windows (Git Bash);  .venv/bin/activate on Mac/Linux
pip install -r requirements.txt
cp .env.example .env               # then add your GROQ_API_KEY
python app.py                      # serves on http://localhost:5000
```

> On a machine behind SSL interception, install with
> `pip install -r requirements.txt pip-system-certs --trusted-host pypi.org --trusted-host files.pythonhosted.org`.

## API

| Method | Path      | Body                          | Returns |
|--------|-----------|-------------------------------|---------|
| POST   | `/submit` | `{ "text", "creator_id" }`    | `content_id`, `attribution`, `confidence`, `label`, `signals` |
| GET    | `/log`    | —                             | most recent audit-log entries |
| GET    | `/health` | —                             | liveness check |

### Example

```bash
curl -s -X POST http://localhost:5000/submit \
  -H "Content-Type: application/json" \
  -d '{"text": "The sun dipped below the horizon, painting the sky in hues of amber and rose. I sat on the porch, coffee in hand, watching the neighborhood slowly go quiet.", "creator_id": "test-user-1"}' | python -m json.tool
```

```json
{
    "content_id": "737be591-15a6-484e-9cb8-2ad57e7d0f15",
    "creator_id": "test-user-1",
    "attribution": "uncertain",
    "confidence": 0.2,
    "label": "[placeholder — Signal 1 only] attribution=uncertain, 40% AI-likelihood ...",
    "signals": {
        "signal1_llm": {
            "ai_likelihood": 0.4,
            "rationale": "Poetic tone and uniform register, but lacks idiosyncratic language.",
            "ok": true
        }
    },
    "status": "classified"
}
```
